# 使用说明
###程序说明
####入口：

- 直接双击，默认调用文件夹下的`template.jpg	`作为源图像
- 在cmd中运行，添加一个图片作为源图像如 
	> sztx.exe template2.jpg </br>
	> sztx.exe d:\tmp\2.jpg
	
####出口:

- 将源图片保存为bmp格式，输出为`dst.bmp	`
- 生成一副图片包含9个部分，依次为：

|R分量|源图片|二值图片|
|:--|:--|:--|
|G分量|灰度图|直方图均衡化后图片|
|B分量|灰度图直方图|直方图均衡化后直方图|
